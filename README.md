The URDB app has a page that lists 10 movies.

The individual show pages for each movie include: 
	movie title, 
	a gif, 
	an embedded trailer, 
	a description 

The app includes the use of a table in the database (migrations, Active Record), controllers, and view pages.

The app uses:
	link_to
	instance variables
	dynamic segments
	named routes
	ActiveRecord
